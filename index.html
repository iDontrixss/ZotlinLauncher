Historial de
versiones.

Release 1.0.0
2025
RELEASE
🎨 Interfaz, Diseño y Experiencia Visual
Pantalla de inicio renovada — limpieza visual profunda, interfaz minimalista enfocada en jugar.
Apartado de ajustes completamente rediseñado con opciones avanzadas, cambio de temas dinámico y organización intuitiva.
Scrollbar personalizada integrada con el modo oscuro y la estética del launcher.
Transiciones fluidas sin congelamientos gracias al nuevo sistema asíncrono — navegación instantánea.
Protección anti-spam en botones: re-ejecución deshabilitada visualmente al hacer clicks rápidos.
⚡ Rendimiento e Ingeniería
Caché inteligente de almacenamiento: cálculo del tamaño de carpetas (16.000+ archivos) realizado una sola vez en memoria.
Optimización del cambio de secciones con caché de nodos (sectionNodeCache) para evitar re-ejecución de scripts.
🚀 Nuevas Funciones
Soporte Premium: autenticación oficial con Microsoft Login.
Java S: runtimes de Java preinstalados para compatibilidad automática con cualquier versión de Minecraft.
i18n completa: soporte bilingüe Español/Inglés con traducción en caliente de toda la interfaz.
🐛 Corrección de Errores
Bug crítico: fuga de memoria por scripts duplicados en cada click (let/const already declared) — corregido.
NeoForge/Modpacks: paracaídas if (!target) return para evitar crashes al cambiar de pestaña durante la carga.
EPERM Fix: reestructuración del limpiador de archivos en main.js para evitar fallos con archivos abiertos por Electron en Windows.
Eliminación de HTML corrupto y bloques JS duplicados en la vista de instancias.
